# Bigger Bolder Baking — Amazon Performance Dashboard

Single-page dashboard mockup for a prospective Amazon client (Gemma Stafford / Bigger Bolder Baking).

## Stack
Plain HTML + inline CSS. No build step, no dependencies. Open `index.html` in any browser.

## Hosting
Deployed via GitHub Pages from the `main` branch root.

## Sections
1. Topline KPIs (Revenue, Units, Sessions, Ad Spend)
2. Catalog Performance by SKU
3. Media Lift · Content to Commerce
4. Media Lift · Channel Performance
5. Bold Baker Reviews
6. Inventory Health
